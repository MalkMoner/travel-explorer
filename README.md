 Travel Explorer - Dynamic Travel & Booking Platform

**Travel Explorer** is a modern and integrated web application for managing and booking tourist trips and hotels. Built using **React** and the fast development tool **Vite**, this project aims to deliver an interactive and seamless user experience (UI/UX) that simulates global travel platforms.

 Repository Structure
To facilitate the project review process, the uploaded files are organized as follows:
* **`src/`**: The main folder containing all project pages, components, hooks, and contexts for state management, payments, and dark mode.
* **`assets/`**: A dedicated folder containing all images, icons, and logos used in the user interfaces.
* **`travel-explorer.pptx`**: The official presentation file covering the project idea, features, and interface walkthrough for the discussion.

---
 Project Objectives
* Provide an interactive and attractive user interface suitable for modern tourism companies.
* Apply State Management concepts and reusable Components in React.
* Simulate a complete and secure electronic booking and payment cycle (Mock Payment).
* Deliver a flexible user experience that supports Dark Mode for eye comfort.

---

 Tech Stack
* **Frontend Framework:** React (Vite)
* **Routing:** React Router DOM (v6)
* **Styling & UI:** CSS3 / Vanilla CSS (with full responsiveness across all screens)
* **State Management:** React Hooks (`useState`, `useEffect`, `Context API`)
* **Notifications:** React Toastify / Custom Toast Notifications

---

 Key Features

1.  **Dynamic Dark Mode:** Seamlessly switch the entire website's theme with a single click for enhanced eye comfort.
2.  **Live Boarding Pass:** During account registration, the ticket data updates instantly in real-time as the user types (Real-time Preview).
3.  **Advanced Filtering:** Categorize trips into (Domestic within Egypt / International) with an instant update of the available trips counter.
4.  **Booking Terminal:** An airport-style simulation terminal featuring a moving flight ticker (Live Ticker) and flexible switching between flight and hotel bookings.
5.  **Secure Mock Stripe Payment:** An interactive and secured payment interface simulating the popular Stripe gateway to experience a full booking cycle without real charges.
6. **Toast Notifications:** Confirmation messages that appear and disappear automatically upon account creation, logging out, and payment confirmation.
